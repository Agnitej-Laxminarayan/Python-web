Dictionaries

A dictionary is a collection which is unordered, changeable, and indexed. In Python, dictionaries are written with curly brackets, and they have keys and values.

Syntax

1# Creating an empty dictionary
2my_dict = {}
3
4# Creating a dictionary with initial values
5my_dict = {"key1": "value1", "key2": "value2"}

Creating a Dictionary

1# Creating a simple dictionary
2my_dict = {"name": "John", "age": 30, "city": "New York"}
3
4# Creating a dictionary with mixed data types
5mixed_dict = {"key1": 123, "key2": [1, 2, 3], "key3": (4, 5, 6)}

Accessing Dictionary Items

You can access the items of a dictionary by referring to its key name, inside square brackets.

1# Accessing items in a dictionary
2print(my_dict["name"])  # Output: John
3print(mixed_dict["key2"])  # Output: [1, 2, 3]

Note

If the key does not exist, attempting to access it will raise a KeyError.

Modifying Dictionary Items

1# Modifying items in a dictionary
2my_dict["age"] = 31
3print(my_dict)  # Output: {"name": "John", "age": 31, "city": "New York"}

Adding Items to a Dictionary

1# Adding items to a dictionary
2my_dict["email"] = "john@example.com"
3print(my_dict)  # Output: {"name": "John", "age": 31, "city": "New York", "email": "john@example.com"}

Removing Items from a Dictionary

 1# Using pop() to remove a specific item
 2my_dict.pop("age")
 3print(my_dict)  # Output: {"name": "John", "city": "New York", "email": "john@example.com"}
 4
 5# Using del to remove a specific item
 6del my_dict["city"]
 7print(my_dict)  # Output: {"name": "John", "email": "john@example.com"}
 8
 9# Using clear() to empty the dictionary
10my_dict.clear()
11print(my_dict)  # Output: {}

Warning

The clear() method will remove all elements from the dictionary. Use it with caution.

Dictionary Methods

Here are some common methods you can use with dictionaries:

  • clear(): Removes all elements from the dictionary

  • copy(): Returns a copy of the dictionary

  • fromkeys(): Returns a dictionary with the specified keys and value

  • get(): Returns the value of the specified key

  • items(): Returns a list containing a tuple for each key-value pair

  • keys(): Returns a list containing the dictionary’s keys

  • pop(): Removes the element with the specified key

  • popitem(): Removes the last inserted key-value pair

  • setdefault(): Returns the value of the specified key. If the key does not exist, insert the key, with the specified value

  • update(): Updates the dictionary with the specified key-value pairs

  • values(): Returns a list of all the values in the dictionary

Example Usage of Dictionary Methods

 1my_dict = {"name": "John", "age": 30, "city": "New York"}
 2
 3# Clear
 4my_dict.clear()
 5print(my_dict)  # Output: {}
 6
 7# Re-initializing for further examples

10# Copy
11copied_dict = my_dict.copy()
12print(copied_dict)  # Output: {'name': 'John', 'age': 30, 'city': 'New York'}
13

16value = 0
17new_dict = dict.fromkeys(keys, value)
18print(new_dict)  # Output: {'key1': 0, 'key2': 0, 'key3': 0}
19
20# Get
21print(my_dict.get("name"))  # Output: John
22
23# Items
24print(my_dict.items())  # Output: dict_items([('name', 'John'), ('age', 30), ('city', 'New York')])
25
26# Keys
27print(my_dict.keys())  # Output: dict_keys(['name', 'age', 'city'])

29# Pop
30my_dict.pop("age")
31print(my_dict)  # Output: {'name': 'John', 'city': 'New York'}

Difference between Lists and Dictionaries

Lists

  • A list is a collection which is ordered and changeable.

  • Lists are written with square brackets [].

  • List items are indexed, with the first item having index [0].

  • Lists allow duplicate members.

  • Elements are accessed using their index positions.

Dictionaries

  • A dictionary is a collection which is unordered, changeable, and indexed.

  • Dictionaries are written with curly brackets {}.

  • Dictionary items are accessed by their key, not by index.

  • Dictionaries do not allow duplicate keys.

  • Keys must be unique and immutable (e.g., strings, numbers, tuples), but values can be of any type.

Key Differences

  1. Order: - Lists maintain the order of items. - Dictionaries do not maintain the order of items (prior to Python 3.7). From Python 3.7 onwards, dictionaries maintain the order of insertion.

  2. Accessing Elements: - List elements are accessed by their index. - Dictionary elements are accessed by their key.

  3. Syntax: - Lists use square brackets []. - Dictionaries use curly brackets {} with key-value pairs separated by colons :.

  4. Duplication: - Lists allow duplicate values. - Dictionaries do not allow duplicate keys.

Example

 1# List example
 2my_list = [1, 2, 3, 4, 5]
 3
 4# Accessing list items
 5print(my_list[0])  # Output: 1
 6
 7# Dictionary example
 8my_dict = {"name": "John", "age": 30, "city": "New York"}
 9
10# Accessing dictionary items
11print(my_dict["name"])  # Output: John
