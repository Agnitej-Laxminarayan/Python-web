Sets

A set is a collection which is unordered, unchangeable (but you can add or remove items), and unindexed. Sets are written with curly brackets.

Creating a Set

1# Creating a simple set
2my_set = {1, 2, 3, 4, 5}
3
4# Creating a set with mixed data types
5mixed_set = {1, "Hello", 3.14, True}
6
7# Creating an empty set
8empty_set = set()

Accessing Set Items

You cannot access items in a set by referring to an index, since sets are unordered. But you can loop through the set items using a for loop, or ask if a specified value is present in a set by using the in keyword.

1# Looping through a set
2for item in my_set:
3    print(item)
4
5# Checking if an item exists in a set
6print(1 in my_set)  # Output: True

Adding Items to a Set

1# Using add() to add an item
2my_set.add(6)
3print(my_set)  # Output: {1, 2, 3, 4, 5, 6}
4
5# Using update() to add multiple items
6my_set.update([7, 8])
7print(my_set)  # Output: {1, 2, 3, 4, 5, 6, 7, 8}

Removing Items from a Set

 1# Using remove() to remove a specific item
 2my_set.remove(2)
 3print(my_set)  # Output: {1, 3, 4, 5, 6, 7, 8}
 4
 5# Using discard() to remove a specific item
 6my_set.discard(3)
 7print(my_set)  # Output: {1, 4, 5, 6, 7, 8}
 8
 9# Using pop() to remove an arbitrary item
10my_set.pop()
11print(my_set)  # Output: {4, 5, 6, 7, 8}
12
13# Using clear() to empty the set
14my_set.clear()
15print(my_set)  # Output: set()

Set Methods

Here are some common methods you can use with sets:

  • add(): Adds an element to the set

  • clear(): Removes all elements from the set

  • copy(): Returns a copy of the set

  • difference(): Returns a set containing the difference between two or more sets

  • difference_update(): Removes the items in this set that are also included in another specified set

  • discard(): Removes the specified item

  • intersection(): Returns a set, that is the intersection of two other sets

  • intersection_update(): Removes the items in this set that are not present in other specified sets

  • isdisjoint(): Returns whether two sets have a intersection or not

  • issubset(): Returns whether another set contains this set or not

  • issuperset(): Returns whether this set contains another set or not

  • pop(): Removes an element from the set

  • remove(): Removes the specified element

  • symmetric_difference(): Returns a set with the symmetric differences of two sets

  • symmetric_difference_update(): Inserts the symmetric differences from this set and another

  • union(): Returns a set containing the union of sets

  • update(): Update the set with the union of this set and others
